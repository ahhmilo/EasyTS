Fix Black Bars
Recommended for laptops
Some systems display black bars on the sides when using a custom resolution. This fix sets the display scaling to Full Panel in the Windows Registry, forcing the screen to stretch the image correctly.

⚠ Requires administrator permissions — a UAC prompt will appear.
⚠ A PC restart is required after applying.
May not work on all systems. Best results on Intel and AMD integrated graphics. NVIDIA users should use NVIDIA Control Panel → GPU Scaling → Full Screen instead.
